# Contributing to blink

Thanks for contributing!

## Structure

```
boards/<board-name>/<framework>/<example-name>/
```

### Example:

```
boards/raspberry-pi-pico/micropython/basic-blink/
```

## Required

Each example should include:

- Source code
- README.md

README should mention:

- Board
- Framework/language
- LED pin
- Flash/upload steps

## Naming

Use lowercase kebab-case:

```
raspberry-pi-pico
esp32-devkit-v1
basic-blink
pwm-fade
```

## Keep Examples
- Minimal
- Beginner-friendly
- Easy to run

Avoid committing build files or binaries.

## Before Submitting
- Test the example if possible
- Keep one concept per example
- Open a pull request

Thanks for helping make blink better 🚀.
