# Blink 💡

The Hello World of embedded systems.

A collection of LED blink examples for different microcontroller boards, frameworks, and programming languages.

## Structure

```
boards/<board-name>/<framework>/<example-name>/
```

### Example:

```
boards/raspberry-pi-pico/micropython/basic-blink/
```
